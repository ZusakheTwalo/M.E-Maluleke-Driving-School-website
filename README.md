# M.E Maluleke Driving School Website

This project is a responsive website developed for **M.E Maluleke Driving School**, a driving school based in Johannesburg, South Africa.

The website provides information about the driving school's services, showcases successful drivers, and allows potential clients to learn more about the school and contact them easily.

## Features

- Modern and responsive design
- Hero landing section
- About section describing the business
- Services offered (Code 8, Code 10, and Code 14 training)
- Success gallery featuring students who obtained their driver's licenses
- Contact section for customer enquiries
- Mobile responsive navigation with toggle menu

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

The purpose of this project is to provide a professional online presence for the driving school and make it easier for potential clients to find information about their services.
